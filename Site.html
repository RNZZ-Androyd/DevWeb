Começar

HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.


<!DOCTYPE html>
<html>
  <head>
    <title>Título da Página</title>
  </head>
  <body>
    <h1>Bem-vindo ao meu site!</h1>
  </body>
</html>

CSS

CSS (Cascading Style Sheets) é a linguagem usada para descrever a apresentação de documentos HTML.


body {
    background-color: #0e0e0e;
    color: white;
}
h1 {
    color: #ff0055;
}

JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web.


function saudacao() {
    alert("Bem-vindo ao meu site!");
}

Boas Práticas

  • Comente seu código.
  • Utilize um estilo de codificação consistente.
  • Teste seu código em diferentes navegadores.
  • Mantenha a semântica do HTML.
